# 复盘页

> 复盘页是 BlinkLife 的未来核心功能，将在回放页基础上融合运动数据分析和 AI 智能复盘，帮助用户从"记录精彩瞬间"升级到"理解运动表现"。

:::info 状态
此功能尚在规划中，以下内容为合理假设，待产品方案确认后更新。
:::

## 与回放页的区别

| 维度 | 回放页 (当前) | 复盘页 (规划) |
|------|-------------|-------------|
| 核心 | 视频回放 + 事件浏览 | 数据分析 + AI 洞察 |
| 时间轴 | 打点 marker | 打点 + 运动数据曲线 |
| 输出 | 剪辑视频片段 | 复盘报告 + 改进建议 |
| 数据源 | .blink 打点文件 | 打点 + 手表传感器 + AI |

## 合理假设

| 假设 | 依据 |
|------|------|
| 复用时间轴组件 | TimelineProgressTrack 三层架构可扩展新 Layer |
| 运动数据来自手表 | WatchCommunicationService 已建立通信通道 |
| AI 分析在服务端 | 客户端资源有限 |

## 待确认项

| 项目 | 说明 |
|------|------|
| 运动数据格式 | 手表传感器数据的采集格式和存储方案 |
| AI 分析模型 | 模型选择、输入输出格式 |
| 复盘报告格式 | 文字 / 图表 / 视频标注 |
| 时间轴扩展方案 | 新增数据层 vs 复用现有层 |
